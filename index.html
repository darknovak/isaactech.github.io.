O que é HTML?

HTML (HyperText Markup Language) é a linguagem que define a estrutura de uma página web. Ela organiza o conteúdo em elementos como cabeçalhos, parágrafos, imagens, e links.

Elementos Importantes do HTML

  • <h1> a <h6>: Tags de cabeçalhos que organizam a hierarquia de títulos.
  • <p>: Define um parágrafo.
  • <a>: Cria links entre páginas. Exemplo: 
    <a href="https://www.google.com">Visite o Google</a>
  • <img>: Insere imagens na página. Exemplo: 
    <img src="imagem.jpg" alt="Descrição da Imagem">
  • <div>: Um contêiner genérico que agrupa outros elementos.

Estrutura Básica de um Documento HTML

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
</head>
<body>
    <h1>Bem-vindo ao Isaac Tech</h1>
    <p>Este é um exemplo básico de HTML.</p>
</body>
</html>

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento HTML. Ela controla a aparência e o layout dos elementos.

Principais Propriedades do CSS

  • color: Define a cor do texto. Exemplo: 
    h1 { color: blue; }
  • background-color: Define a cor de fundo de um elemento. Exemplo: 
    body { background-color: lightgrey; }
  • font-family: Especifica a fonte do texto. Exemplo: 
    p { font-family: 'Arial', sans-serif; }
  • margin: Define o espaço ao redor dos elementos. Exemplo: 
    div { margin: 20px; }
  • padding: Define o espaço interno de um elemento. Exemplo: 
    div { padding: 15px; }

Exemplo de CSS Completo

body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}

h1 {
    color: #4CAF50;
    text-align: center;
}

p {
    color: #333;
    font-size: 18px;
    line-height: 1.6;
}

.container {
    width: 80%;
    margin: auto;
    padding: 20px;
    background-color: white;
    box-shadow: 0px 0px 10px rgba(0, 0, 0, 0.1);
}

O que é JavaScript?

JavaScript é uma linguagem de programação que permite adicionar funcionalidades dinâmicas e interatividade a uma página web. Com JS, você pode modificar o HTML e CSS em tempo real.

Conceitos Básicos de JavaScript

  • Variáveis: Armazenam dados. Exemplo: 
    let nome = "Isaac";
  • Funções: Executam uma ação. Exemplo: 
    function saudacao() {
    alert("Olá, " + nome);
}
  • Eventos: Interagem com o usuário. Exemplo: 
    document.getElementById("meuBotao").addEventListener("click", saudacao);

Exemplo Completo de Interatividade

<button id="meuBotao">Clique aqui</button>

<script>
let nome = "Isaac";

function saudacao() {
    alert("Olá, " + nome);
}

document.getElementById("meuBotao").addEventListener("click", saudacao);
</script>
